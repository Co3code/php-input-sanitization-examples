# PHP Input Sanitization Examples

This repository provides a collection of PHP code examples that show how to properly sanitize and validate user inputs. These practices help protect your website from common security issues like XSS (Cross-Site Scripting), SQL Injection, and other types of attacks.

## Examples Included:

- **String Sanitization** (Prevent XSS)
- **Email Sanitization**
- **URL Sanitization**
- **Integer Sanitization**
- **Float Sanitization**
- **Phone Number Sanitization**
- **Username Sanitization**
- **Date Validation and Sanitization**
- **SQL Injection Prevention using Prepared Statements**
- **JSON Encoding for JavaScript Context**
- **Search Query Sanitization**
